# Roadmap for micael

## v0.1.0: Europa

* Memory Management:
  * Kernel Heap
  * Page Cache
  * Slab Allocator
* Preemptive Multitasking
* Initial ramdisk
* VFS
   * devfs
* System Console

## v0.2.0: Hyperion

* Symmetric Multiprocessing
* Priority-based scheduling
* 64-bit support (x86_64 port)

## v0.3.0: Pandora

Drivers

* PCI
* SATA
* RTL8139
* PC speaker driver

## v0.4.0: Harpalyke

* Networking

## v0.5.0: Calypso

File system stuff

* Block Device Subsystem
* Pseudo filesystems
   * tmpfs
   * procfs
   * sysfs
* ext2
* rafael
* Custom boot loader

## v.0.6.0: Kerberos

* Crypto

## v0.7.0: Himalia

* dlibc
* Core utils
* Proper shell

## v1.0.0: Titania
Go public!
